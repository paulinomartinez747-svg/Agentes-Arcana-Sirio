# Método Paulino — Psicología Esotérica

## Marcos teóricos
- **Carl Jung:** Arquetipos, inconsciente colectivo, sincronicidad, sombra
- **Alejandro Jodorowsky:** Psicomagia, actos simbólicos, tarot
- **Bert Hellinger:** Constelaciones familiares, órdenes del amor, transgeneracional
- **Gestalt:** Aquí y ahora, integración de polaridades
- **EMDR:** Desensibilización de traumas

## Mapeo planetas → arquetipos
- Sol → El Self, identidad consciente
- Luna → Anima/Animus, cuerpo emocional, madre
- Mercurio → El Trickster, comunicación
- Venus → El Amante, valores, atracción
- Marte → El Guerrero, impulso, acción
- Júpiter → El Sabio, expansión, sentido
- Saturno → El Senex, límites, padre, karma
- Urano → El Rebelde, ruptura, innovación
- Neptuno → El Místico, disolución, trascendencia
- Plutón → La Sombra, muerte-renacimiento, poder

## Estructura de interpretación (para cualquier carta/sesión)
1. **Patrón Central** — 1-2 configuraciones dominantes
2. **La Sombra** — Lo que el nativo no ve de sí mismo
3. **El Don** — La fortaleza escondida en el aspecto difícil
4. **Acto Psicomágico** — Una acción simbólica concreta para integrar

## Principios de sanación
- Integral: cuerpo físico, emociones, mente, energía, espíritu
- La limpieza precede a la protección, y esta a la restauración
- El paciente es protagonista activo, no objeto pasivo
- "Esto no es magia. Es acción simbólica que habla al inconsciente."
